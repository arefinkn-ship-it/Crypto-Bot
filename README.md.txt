# 🚀 Crypto Bot V2

Advanced crypto trading signal bot with multi-strategy scoring, backtesting, and real-time alerts.

## 📋 Features

- **10+ Indicators**: EMA, RSI, MACD, SuperTrend, ADX, Bollinger Bands, VWAP, WaveTrend, ATR, OBV
- **4 Core Strategies**: Trend Following, Breakout, MA Crossover, Mean Reversion
- **Smart Scoring**: Weighted combination with confidence levels
- **Real-time Alerts**: Telegram notifications with rich formatting
- **Backtesting**: Historical performance validation
- **TimescaleDB**: Optimized time-series data storage
- **Docker**: Easy deployment
